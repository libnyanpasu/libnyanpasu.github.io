# Add Icons to Proxy Groups

:::caution[Note]
Icons are only supported by `Mihomo` and `Clash Rust` kernels. `Clash Premium` does not support them.
:::

Uses icon sets from [Koolson/Qure](https://github.com/Koolson/Qure) and flag icons from [HatScripts/circle-flags](https://github.com/HatScripts/circle-flags).
